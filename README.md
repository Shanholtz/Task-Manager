# Task-Manager
Kanban style task board allowing for visualizing and managing tasks.

To run the program locally:

Ensure Node.js and npm are installed
('node -v' and 'npm -v' are alid commands to test)

Extract the ZIP from the github repository and open folder in code editor or terminal.

Run the command 'cd task-board' to change the directory to the project file. 

Then run 'npm install' to install dependencies.
